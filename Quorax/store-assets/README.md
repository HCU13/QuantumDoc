# Store Assets — Quorax (Math Module)

Localized App Store Connect / Play Console metadata for the 5 supported locales.

## Files
- `metadata-en.md` — English (primary, US market)
- `metadata-tr.md` — Türkçe (Turkey)
- `metadata-es.md` — Español (Latin America + Spain)
- `metadata-ar.md` — العربية (MENA)
- `metadata-hi.md` — हिन्दी (India)

## Character limits (App Store Connect)
- Title: **30** chars
- Subtitle: **30** chars
- Promotional Text: **170** chars (editable anytime without review)
- Keywords: **100** chars, comma-separated
- Description: **4000** chars

## Character limits (Play Console)
- App name: **30** chars
- Short description: **80** chars
- Full description: **4000** chars

## Screenshot strategy (per locale)
1. Hero: handwritten math problem being solved via camera
2. Step-by-step with KaTeX fraction/integral
3. Verify My Work — unique selling point
4. Topic analysis dashboard
5. Multi-language demo (show device language active)
